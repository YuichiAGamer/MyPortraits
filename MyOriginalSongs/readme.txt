このフォルダには、私が考案したゲームや小説の世界をイメージして作成した楽曲のうちの 4 曲が収録されています。
以下に、各曲に関する解説を付記します。

***********
2. 各曲解説
***********

1. Blood Princess

ハープシコードを主旋律に使った曲です。以下のようなイメージで、作曲を行いました。

――人も近寄らなくなり久しい、寂れた古城の奥深く。
そこでは自らのベッドである棺桶から、吸血鬼の王女が時折目覚める。
その身を吸血鬼にやつして以来、年を取ることを辞めてしまった彼女は、古城のホールで一人踊る。
その孤独な踊りの中、少女は不死者の心に何を思い浮かべるのか――？

(利用ソフトウェア: Studio One 3 x64, Any Audio Converter)

2. Under The Breeze Of Ether

私の構想しているオリジナル小説に登場する架空の都市、「イーサー・ウインド」のテーマソングとして作曲しました。
この曲の作曲時は、音楽理論の独学もあやふやだったため、ひたすらにノートを重ねて、心地よく感じる響きやコード進行を手探りで探して、何とか形にした一曲です。

――あまたの次元との交差点となる世界、「ジャンクション・ワールド」。
そのジャンクション・ワールドの一つが、イーサー・ウインドと呼ばれる世界である。
イーサー・ウインドは、都市とその上空に満ち溢れる魔法エネルギー「エーテル」の循環でその構造を支えている。
エーテルの循環を制御するのは、都市の上空に浮く「天星輪」――黄道十二星座の力を秘めた巨大な魔力の円環である。
霊気の風の流れの中から、今日もこの都市に次元の異邦人が訪れ、何かが始まろうとしている――。

(利用ソフトウェア: Studio One 3 x64, Any Audio Converter)

3. 電子世界のストレンジャー

私が以前 Visual Basic を利用して作成しようと考えた、あるゲームの BGM に利用することを想定して作曲した曲です。
この 4 曲の中では最も古い曲であり、 DTM を覚え始めの頃に作られた曲であり、音の数は少なく、アンビエント風の仕上がりになっています。

――ありとあらゆる要素が「0」と「1」で構成される、電子世界。
本来ならば、デジタル情報で構成されていない存在は、この世界を訪れることができない。
その世界にふとしたきっかけで迷い込んでしまった者達は、物質で構成された世界とは異なる法則で成り立つこの電子世界において、異邦人として彷徨う――。

(利用ソフトウェア: Studio One 3 x64, Any Audio Converter)

4. 人の子守りし黄金龍

テーブルトーク RPG 「ダンジョンズアンドドラゴンズ(D&D)」のキャンペーンにおいて、ある伏線に利用することを目的として作曲した曲です。 D&D 世界の吟遊詩人が、街角でリュートを手に歌い上げるイメージで作曲したため、あえてカントリーギターとオーボエのみを使い、シンプルかつ素朴な曲調としました。また、この曲のみシンセサイザーで打ち込みを行い、歌詞を付けてあります。

――かつてこの地を恐怖のどん底に陥れた、「髑髏の王」「髑髏の女王」と呼ばれる強大な魔物。
かの魔物を打ち払ったのは、人間の勇者ではなく、黄金の鱗を持つ一頭のドラゴンだった。
「アウルム」と呼ばれるそのドラゴンは、「髑髏の王」と「髑髏の女王」を討つと共に、彼らの真の名前を決して呼んではならない、と人間に言い含めた。その理由は今なお、歴史の彼方に忘れ去られ、明らかではない――。

(利用ソフトウェア: Studio One 3 x64, Any Audio Converter)
(利用ハードウェア: Yamaha Reface DX)
