1. 概説
*******

このファイルには、 Visual Basic 2010 を利用して作成された、
「TriCryptTest」の実行ファイル、解説 PDF 、およびソリューションファイルが同梱されています。
なお、 "TriCrypt" 法とは、筆者が独自に考案したバイナリデータの暗号化方法です。


TriCryptTest		Visual Studio 2010 を利用して作成された TriCryptTest のソリューションファイルです。
			TryCryptTest.exe のソースコードが含まれています。

TriCryptTest.exe	TryCryptTest の実行ファイルです。

readme.txt		このファイルです。

解説.pdf		TriCrypt の内容や操作方法などをまとめた PDF ファイルです。
