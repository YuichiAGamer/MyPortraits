1. 概説
*******

このファイルには、 html5 の Canvas を利用して作成された、ライフゲーム起動用の html ファイル、
およびその実行のためのライブラリ (JavaScript製)が同梱されています。

index.html			Canvas のみ存在する単純な html ファイルです。
				Web ブラウザで開くと、 "Starting Life Game" というダイアログが開き、
				その後ライフゲームが開始されます。
				(スマートフォン上での起動を想定した画面設計ですが、 
				Firefox や Google Chrome などのPC 用 Web ブラウザでも動作します)

lib				index.html から呼び出される、 Javascript の自作ライブラリが置かれています。

lib/MasterInclude.js		今後のライブラリの追加・拡張を想定して、
				どのライブラリを呼び出すかを一元管理するライブラリです。

lib/CellGrid/CellGrid.js	"lib/MasterInclude.js" 経由で "index.html" に呼び出されるスクリプトファイルです。
				ライフゲームにおけるセルの生死判定のメソッドなどが実装されています。
			
readme.txt			このファイルです。

*********************
2. ライフゲームとは？
*********************

ライフゲームとは、 1970 年にイギリスの数学者が考案した、シミュレーションゲームです。

セル・オートマトンという、正方形の格子で区切られた空間の中で、
その格子ひとつひとつが次の世代で「生きる」か「死ぬ」か「新たに生まれるか」を判定し、
その結果を反映する、という動作を繰り返します。
その様子が、あたかも生きている生物の細胞のように見えるため、このような名前が付いたとされます。

以下のリンク先などに詳細がありますので、必要に応じてご確認ください。

ライフゲーム
https://ja.wikipedia.org/wiki/%E3%83%A9%E3%82%A4%E3%83%95%E3%82%B2%E3%83%BC%E3%83%A0

セル・オートマトン
https://ja.wikipedia.org/wiki/%E3%82%BB%E3%83%AB%E3%83%BB%E3%82%AA%E3%83%BC%E3%83%88%E3%83%9E%E3%83%88%E3%83%B3

筆者は上述されたライフゲームの規則に加え、以下のような独自実装を行っています。

i). セルの数は、縦 20 個×横 30 個の、合計 600 個とする。

ii). これら 600 個のセルは、ライフゲーム開始の時点で、
それぞれ 1/2 の確率で生きており、 1/2 の確率で死んでいる、とランダムに設定される。

iii).  Canvas 上の任意のセルをクリック(タップ)すると、
クリックしたセルおよび、その右・下・右下にある合計 4 つのセルを手動で生きている状態に設定できる。

※ ライフゲームの規則上、クリックした 1 セルのみを生き返らせる or 3 × 3 以上の広範囲のセルを生き返らせると、
次の世代で直ちに「孤独死」/「過密死」が起きてしまうため、生き返るセルの数は 2 × 2 の 4 つと実装しました。
