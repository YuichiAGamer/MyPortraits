*******
1. 概説
*******

このフォルダには、筆者が Unity の利用方法を学習する目的で作成した「UFO Game」の実行ファイルと、そのソースファイルが含まれています。
なお、 UFO Game の作成方法については、以下のリンク先にあるチュートリアルを参考としました。

https://unity3d.com/jp/learn/tutorials/projects/2d-ufo-tutorial

UFOGame		UFO Game のソースファイルです。

UFOGame(Build).zip	上記ソースファイルをビルドして作成された、 UFO Game の実行ファイルです。解凍すると、中に実行ファイルが存在します(Github のファイルアップロード上限の関係で、 zip 形式で保存してあります)。

readme.txt		このファイルです。

********************************
2. UFO Game の起動方法
********************************

同梱の UFOGame(Build).zip ファイルを適当な方法で解凍し、中にある UFOGame.exe を実行してください。なお、この実行ファイルは Windows 向けのビルドとなっています。 UFO は、 W, A, S, D (もしくは↑、←、↓、→)キーで、対応する方向に動かすことができます。

********************************
3. 「Cosmic Barrage!」の終了方法
********************************

UFO Game の右上に出てくる「×」ボタンをクリックしてください。
